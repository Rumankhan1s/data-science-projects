# Titanic Survival Prediction

This project predicts whether a passenger survived the Titanic disaster using machine learning.

## Dataset
Titanic dataset containing passenger details like age, gender, ticket class, and fare.

## Machine Learning Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Best Model
Random Forest achieved the highest accuracy (~81.5%).

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
